Chatwork LM プライバシーポリシー

最終更新日: 2026-02-21

1. 取得する情報

本拡張機能は、以下の情報を取得・保存する場合があります。

  • ユーザーが入力した Chatwork API トークン
  • Chatwork メッセージ情報（本文、送信者名、送信日時、ルーム情報）
  • ユーザーが入力した質問文、検索期間などの操作情報
  • 拡張機能の表示設定・利用に必要なローカル設定情報

2. 利用目的

  • Chatwork メッセージの同期・検索・要約表示
  • 質問への回答生成と根拠メッセージ提示
  • 利用体験の維持（再入力の削減、表示状態の保持）

3. 外部送信

本拡張機能は機能提供のため、必要最小限の情報を以下に送信する場合があります。

  • Chatwork API（メッセージ取得のため）
  • 開発者が管理するプロキシサーバー（OpenAI API連携のため）
  • OpenAI API（回答生成のため）

4. 保存場所と期間

  • 拡張機能設定・同期データは、主にユーザーのブラウザ内ストレージ（chrome.storage）に保存されます。
  • 保存期間はユーザーの利用状況に依存し、ユーザーが拡張機能のデータ削除またはアンインストールを行うまで保持される場合があります。

5. 第三者提供

法令に基づく場合を除き、取得情報を販売・貸与・不正な目的で第三者提供しません。

6. セキュリティ

通信は HTTPS を利用し、取り扱う情報は目的に必要な範囲に限定します。

7. ユーザーによる管理

  • ユーザーは拡張機能設定の変更により、一部データの更新・削除を行えます。
  • 拡張機能を削除することで、ブラウザ内データは削除対象となります。

8. 改定

本ポリシーは必要に応じて改定される場合があります。重要な変更時は、本ページの更新日を変更して告知します。
